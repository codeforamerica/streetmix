Error codes
-----------

These are error codes that may occur while using Streetmix.

+-------------+-------------------------------+-------------------------------+
| Error code  | Reason                        | Suggested fix                 |
|             |                               |                               |
+=============+===============================+===============================+
| **9B**      | **Data error**: The server    | Load a different street.      |
|             | sent street data that had     |                               |
|             | no data in it.                |                               |
